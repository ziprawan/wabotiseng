Getting Started
---------------

Langsung aja kirim perintah ke bot atau tambahin ke grup chat, dan yaudah. Gitu aja.

Formatnya gini (ga pake tutup kurung):

   ``(prefix)(perintah) (argumen)``

Contoh: 
   
   ``.help``

Prefix yang digunakan di bot ini adalah ``.`` dan ``/``
